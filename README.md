# FIRE Simulator

Monte Carlo retirement simulator for ISA, taxable investments, pension, and cash buffer with configurable withdrawal strategies and UK tax handling.

## Quick Start

```bash
cargo run -- serve 8080
```

Open:

- `http://127.0.0.1:8080/`

## Full Documentation
